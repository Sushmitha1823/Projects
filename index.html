About

Beaches in Bharath is a web platform that allows users to explore beaches across India through an interactive map interface. Users can select a state to view information about beaches, including history, culture, and attractions. The site features a chatbot for instant queries and a data analytics section showing beach popularity, accidents, and safety alerts. This project promotes coastal tourism, preserves beach heritage, and ensures visitor safety with real-time insights and assistance.

India, with its vast coastline, is home to some of the most stunning beaches, offering scenic beauty, adventure sports, and a glimpse into coastal culture.
